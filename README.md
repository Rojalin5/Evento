# Evento 🎟️

Scalable event booking backend built with NestJS.

## Features

- User Registration & Login
- JWT Authentication
- Role Based Authorization
- Event Management
- Ticket Booking System
- Booking History
- Admin Controls
- API Documentation with Swagger
- Validation & Error Handling

## Tech Stack

- NestJS
- TypeScript
- MongoDB
- Mongoose
- JWT
- Passport
- bcrypt
- Swagger
